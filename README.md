Healthcare Analysis Dashboard (Excel Project)
Project Overview

This project focuses on analyzing healthcare data to gain insights into patient demographics, hospital visits, treatment patterns, and overall healthcare utilization. The goal is to transform raw data into meaningful insights using Excel.

Objective

To analyze patient and hospital data
To track key healthcare metrics like visits, treatments, and test rates
To create an interactive dashboard for better decision-making

Tools & Technologies Used
Microsoft Excel
Data Cleaning
Pivot Tables
Charts & Visualizations
Slicers & Interactive Dashboard

Dashboard Features
Patient Overview (Total Patients, Visits, Doctors)
Visit Analysis (Monthly trends, Visit status)
Patient Demographics (Age Group, Gender Distribution)
Treatment Analysis (Medication, Surgery, Therapy, Vaccination)
Test Analysis (X-Ray, MRI, CT Scan, Blood Test)
Doctor Performance (Top doctors by visits)

Key Insights
Majority of patients fall under the 36–60 age group
Medication and Vaccination are the most common treatments
Peak visits observed during specific months indicating seasonal trends
Certain tests like CT Scan and X-Ray are highly utilized
Balanced gender distribution among patients
